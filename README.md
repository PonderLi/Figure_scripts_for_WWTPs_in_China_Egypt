# Cross-regional Metagenomic Insights into Clinical and Stable Resistomes in Urban Wastewater Systems

This repository provides the bioinformatic workflows and plotting scripts used in the manuscript **“Cross-regional Metagenomic Insights into Clinical and Stable Resistomes in Urban Wastewater Systems.”** The project investigates clinical and stable antibiotic resistance genes in urban wastewater treatment plants from Egypt and China using shotgun metagenomic sequencing. The code includes ARG profiling, stable ARG screening, host and mobile genetic element association analyses, putative HGT analysis, statistical testing, and visualization of some figures.

For most figures, Chiplot (https://www.chiplot.online/) was applied for drawing. The Figure 1C was used R, the demo code and file was uploaded.
